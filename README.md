# Freelancer KPI System

A web-based Freelancer KPI Tracking and Analytics System developed using HTML, CSS, JavaScript, LocalStorage, Chart.js and SheetJS.

## Technologies Used

- HTML
- CSS
- JavaScript
- LocalStorage
- Chart.js
- SheetJS (xlsx)

## Features

- Dashboard
- Project Management
- Task Management
- Freelancer Management
- Reports
- KPI Analytics
- Excel Import
- CSV Export
- On-Time Delivery Tracking
